다음 프로젝트의 1차 MVP를 구현하기 위한 Chrome 확장 프로그램 코드를 작성해줘.

[목표]
ChatGPT 웹사이트에서만 동작하는 간단한 확장 프로그램을 만든다.

[기능]
1. ChatGPT 페이지에서만 실행
2. 입력창을 감지
3. 입력창 옆에 작은 버튼(✨) 추가
4. 버튼 클릭 시:
   - 현재 입력된 텍스트를 콘솔에 출력
   - (아직 API 연결 없음)

[요구사항]
- Manifest V3 사용
- content script 기반으로 구현
- 버튼이 중복 생성되지 않도록 처리
- 코드에 주석 포함

[출력 형식]
1. 폴더 구조
2. manifest.json
3. content.js
4. content.css
5. 실행 및 테스트 방법

초보자가 그대로 따라할 수 있도록 설명 포함해서 작성해줘.
