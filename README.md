# 🧠 Prompt Coach

> 💡 생성형 AI 입력창에서 프롬프트를 클릭 한 번으로 개선해주는 Chrome 확장 프로그램

---

## 📌 소개

Prompt Coach는 사용자가 ChatGPT 등 생성형 AI에서 작성한 프롬프트를 분석하고,  
더 명확하고 구체적인 형태로 개선해주는 도구입니다.

---

## ⚠️ 문제

- 프롬프트를 어떻게 써야 할지 모름
- 질문이 모호해서 결과 품질이 낮음
- 반복 수정으로 시간 낭비

---

## 🎯 해결

> 입력창에서 바로 프롬프트를 개선하는 확장 프로그램

- 복붙 필요 없음
- 클릭 한 번으로 개선
- 사용 흐름 유지

---

## 🚀 핵심 기능

- ✨ 입력창 옆 버튼 추가
- 🧠 프롬프트 문제점 분석
- 🔧 개선된 프롬프트 제안
- 🔄 유지 / 적용 선택

---

## 🔄 사용 흐름

1. 프롬프트 입력
2. 버튼 클릭
3. 개선 결과 확인
4. 적용 or 유지
